    What this demo is and isn't

    Real forward pass of Xenova/distilgpt2 in your browser, ~85 MB cached after first load. We extract residual-stream snapshots at each layer (via output_hidden_states) and attention weights (via output_attentions). The "logit lens" projection uses the same shortcut as the residual-stream demo: dot-product against a 6,000-token chunk of the embedding table, top-1. So each cell's prediction is exact for those 6,000 tokens, and approximate (but usually correct) for the rare ones. The "MLP delta" shown is the residual-stream change between adjacent blocks, which conflates attention + MLP within the same block — see the long-form post above for the cleaner per-component decomposition.
