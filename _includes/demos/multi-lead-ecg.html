Interactive · Reading the same heartbeat from four angles

Four ECG leads, same heart, same instant. Each lead is a different vantage point — and each disagrees a little. Click a diagnosis to see how the leads shift, and where the model looks.

How this demo works

Each track shows a stylized ECG waveform for the selected condition, from one of four leads (II, V1, V5, aVF). The colored spans mark which regions the paper's per-lead attention block weights most heavily for the diagnosis. Single-lead classifiers see just one row; the fusion block reads all four and their disagreements.
