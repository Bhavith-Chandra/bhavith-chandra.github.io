What's actually happening inside each station

1. Tokenizer splits text into numeric IDs (often subword pieces). 2. Embedding looks up a vector for each ID from a big table. 3. Attention lets every position peek at every other position and pull in what's relevant. 4. MLP is a per-position feed-forward network that transforms the vector non-linearly. 5. Repeat stacks attention+MLP dozens of times. 6. Unembedding projects the final vector into a probability over the vocabulary. The belt you see is the residual stream, the running total that every block reads from and writes to.
