Interactive · Attention over a source sentence

Every generated summary token gets to look back at the whole source. The heatmap below shows where a real attention head focuses when producing each output word. Click an output token.

How this demo works

Six source tokens on top, six output tokens on the bottom. When you click an output token, the arcs show that token's attention distribution over the source. The bottleneck toggle mimics a vanilla encoder-decoder: one fixed vector for the whole source, so every output token has to squeeze information through the same channel. That's what the paper's coverage loss is fighting against.
