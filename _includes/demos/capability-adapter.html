How this demo works

Each analysis declares a set of required capabilities. Each architecture's adapter declares which capabilities it supports. The harness resolves compatibility at load time — a green result means the analysis runs, a red result names exactly which capability the model doesn't have. This is the type-system check that keeps analyses honest across architectures.
